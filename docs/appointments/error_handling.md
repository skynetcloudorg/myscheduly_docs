# Errors and Troubleshooting

Version: 1.0.29 | Last updated: 2025-11-27

## Common Error Messages
- No available time slots: pick a different period or use Custom Time.
- Please select date, service, and staff member first: complete Step 1.
- Time slot validation failed: re-enter Custom Time and validate again.
- Invalid promo code: check the code or try another one.
- Time too close to now: choose a later slot (about 30 minutes buffer).
- Closed or holiday: pick another date or use Custom Time if recording history.
- Cannot edit finished appointments: update before marking Finished.

## Troubleshooting Guide
- If slots are empty:
  - Confirm Date, Service, and Professional are selected.
  - Try a different time period (Morning/Afternoon/Evening).
  - Use Custom Time and validate.
- If reminders didn’t send:
  - Ensure status is Confirmed and contact info is correct.
  - Ask customers to check spam folders.
  - Confirm your calendar and email/SMS settings are active.
- If filters don’t update:
  - Clear search and click Reset.
  - Refresh the page.
